# Transit-Detect
A repository to focus only on finding transit of RV confirmed and candidate planets. Constraints: Orbital Period: [0.5, 33.0] days

## Downloading Data
Required targets can be found in either NASA Exoplanet Archive or The Extrasolar Planets Encyclopaedia. Download the .csv files. Include the TIC list for confirmed planets if you are downloading it from NASA Exoplanet Archive. In case you use The Extrasolar Planets Encyclopaedia, you can use TICconverter.py.
